# <h3>🧮 Simple Calculator<h3>

This is a simple calculator built using **HTML**, **CSS**, and **JavaScript**. I created this project while learning front-end web development to improve my understanding of JavaScript, DOM manipulation, and responsive layouts.

The calculator can perform basic arithmetic operations like addition, subtraction, multiplication, and division. Although it's a beginner project, it helped me understand how HTML, CSS, and JavaScript work together to build interactive web applications.

## ✨ Features

* Basic arithmetic operations (+, −, ×, ÷)
* Decimal number support
* Clear (C) button
* Simple and clean user interface
* Responsive button layout using Flexbox

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/dikshit0012/calculator.git
```

2. Open the project folder.

3. Double-click `calculator.html` (or open it in your browser).

That's it! No additional setup or installation is required.
